# Privacy-Preserving Protocols for Machine Learning: Design and Application to Neural Network Training



### Requirements
---
* The code should work on most Linux distributions (It has been developed and tested with [Ubuntu](http://www.ubuntu.com/) 18.04).

* **Required packages for P3ML:**
  * [`g++`](https://packages.debian.org/testing/g++)
  * [`make`](https://packages.debian.org/testing/make)
  * [`libssl-dev`](https://packages.debian.org/testing/libssl-dev)
  * [emp-toolkit] (https://github.com/emp-toolkit/emp-tool.git)
  * [emp-ot] (https://github.com/emp-toolkit/emp-ot/tree/v0.1)
  * [Eigen3] (https://gitlab.com/libeigen/eigen/-/releases)

  Install these packages with your favorite package manager, e.g, `sudo apt-get install <package-name>`.


#### Building the code

To build Falcon, run the following commands:

```
git clone 
cd P3ML
make terminal
```
